UltraStack User Guide

UltraStack is a casual mobile game where the player attempts to stack blocks as high as possible. I finished development and released the game on the Apple App Store and Google Play Store in early 2019. The app has been archived and is no longer available, but you can download the PC version below. It was developed using Unity3D and C#.

Installation

Follow the build instructions on GitHub or download the executable here.

How to Play

The goal of UltraStack is to create the tallest tower possible. A player's score is equal to the tower's height. However, the game utilizes the Havoc physics engine, making it possible for an unstable tower to fall over. If the tower falls over, the game ends. The most successful towers have a balance of stability and height.

Controls

Use the left and right arrow keys to move a block left and right, respectively. To make a block drop faster, hold the down arrow key. To rotate a block, tap the Z key.
