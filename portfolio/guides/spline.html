Spline Curve Editor/Interpolator User Guide

The spline curve editor is an interactive way to learn about Hermite and Catmul-Rom splines. The application allows you to add and modify control points.

Installation

Follow the build instructions on GitHub or download the executable here.

Adding Points

GIFs

To add points to the curve, enter add mode. Add mode can be selected from the UI in the top left corner. Once in add mode, click the desired location of the new point.

Editing Points

GIFs

When in edit mode, points on the curve can be moved. To enter edit mode, select edit from the mode drop down. Simply click and drag points to move them.

Control Points

GIFs

Click the Ctrl pts button in the UI to display control points. These can be adjusted/moved when in edit mode.

Clearing Points

GIFs

To reset and clear all points in the curve, click the Clear button in the UI.
