Inverse Kinematics Demos User Guide

The spline curve editor is an interactive way to learn about Hermite and Catmul-Rom splines. The application allows you to add and modify control points.

Installation

Follow the build instructions on GitHub or download the executable here.

Controls

Rotate Camera: Left Click + Drag
Move Camera: Right Click + Drag
Zoom Camera: Middle Mouse + Drag
Reset Camera: F Key
Move Two-Link Target Point: Left Ctrl + Left Click + Drag on target point

Reset Two-Link Pose

GIFs

To reset the position of the two link, click the Reset Pose button.

Set Two-Link Target Position

GIFs

Use the X, Y, and Z fields under the Goal position tab to directly set the target's position.
